# Ozon Add-on for Home Assistant

Интеграция с Ozon для получения избранных товаров и сохранения их в локальную базу данных.

## Описание

Этот add-on подключается к Ozon, получает список избранных товаров и сохраняет их название и цену в локальную базу данных. Данные обновляются каждый час.

## Установка

1. Добавьте репозиторий в Home Assistant:
   - Перейдите в **Настройки** → **Дополнения** → **Репозитории**
   - Добавьте: `https://github.com/wargotik/wargot-ha-addons`
   - Нажмите **Добавить**

2. Установите add-on:
   - Перейдите в **Настройки** → **Дополнения**
   - Найдите **Ozon** в списке
   - Нажмите **Установить**

## Конфигурация

После установки перейдите на вкладку **Конфигурация** и укажите:

- **username** - Логин от вашего аккаунта Ozon
- **password** - Пароль от вашего аккаунта Ozon

После заполнения нажмите **Сохранить** и перезапустите add-on.

## Использование

1. Запустите add-on через вкладку **Информация**
2. Проверьте вкладку **Журнал** для просмотра логов работы
3. Данные сохраняются в файл `/data/ozon_storage.json`

## Логи

Все логи работы add-on доступны на вкладке **Журнал**. Там вы можете увидеть:
- Процесс подключения к Ozon
- Количество полученных избранных товаров
- Ошибки, если они возникают

## Хранилище данных

Данные о избранных товарах сохраняются в файл:
```
/data/ozon_storage.json
```

Формат данных:
```json
{
  "favorites": [
    {
      "name": "Название товара",
      "price": 1000,
      "id": "1"
    }
  ]
}
```

## Обновление

Add-on автоматически обновляет данные каждый час. Вы также можете перезапустить add-on для принудительного обновления.

## Поддержка

При возникновении проблем проверьте:
1. Вкладку **Журнал** на наличие ошибок
2. Правильность введённых учётных данных
3. Доступность интернет-соединения
